## Dataset Access

The **UniGeoRS Dataset** is available for **academic research purposes only**.
Researchers who would like to access the dataset are required to submit a signed **Data Use Commitment Letter**.

### How to Apply

Please follow the steps below to request access to the UniGeoRS Dataset:

1. Download the Data Use Commitment Letter:

   [UniGeoRS_Data_Use_Commitment.pdf](./docs/UniGeoRS_Data_Use_Commitment.pdf)

2. Fill in the required information in the document, including:

   * Name
   * Affiliation / Institution
   * Email
   * Signature
   * Date

3. Send the signed PDF file to the dataset owner by email.

### Email Requirement

Please use the following email format when applying for access.

**Recipient:**

```text
liangxiao@bit.edu.cn
```

**Email Subject:**

```text
[UniGeoRS Dataset Request] Your Name - Your Institution
```

**Email Body:**

```text
Dear UniGeoRS Dataset Team,

I would like to apply for access to the UniGeoRS Dataset for academic research purposes.

My information is as follows:

Name:
Affiliation / Institution:
Email:
Research Topic / Purpose:
Intended Use of the Dataset:

I have read and signed the Data Use Commitment Letter, and the signed PDF file is attached to this email.

I confirm that the UniGeoRS Dataset will be used only for academic research and non-commercial purposes.

Best regards,
Your Name
```

**Required Attachment:**

```text
Signed_UniGeoRS_Data_Use_Commitment_YourName.pdf
```

After receiving the signed commitment letter, we will review the application and provide access instructions if the request is approved.
